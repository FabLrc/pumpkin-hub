Pumpkin Hub

Le registre communautaire de plugins pour Pumpkin MC — performant, sécurisé, open source.

À propos

Pumpkin Hub est la plateforme centrale de distribution et de gestion de ressources (plugins, scripts, maps) pour le serveur Minecraft Pumpkin MC. Contrairement aux plateformes Java classiques, Pumpkin Hub est conçu pour la performance brute et la sécurité des binaires Rust.

Status
Le projet est actuellement en Phase 1 — Core Platform. L'infrastructure de développement est opérationnelle ; les bases de données et l'authentification sont en cours d'implémentation.

Fonctionnalités clés

📦 Registre de plugins
Catalogue complet avec recherche full-text, filtres par catégorie, architecture CPU et version Pumpkin.
🔒 Sécurité native
Checksums SHA-256, scan automatique des binaires et signature cryptographique des releases.
🚀 Performance Rust
API backend en Rust/Axum avec middleware stack complet, conçue pour des temps de réponse sub-milliseconde.
🔗 Intégration GitHub
Authentification OAuth, publication automatique via tags/releases et synchronisation des README.
🔍 Recherche instantanée
Powered by Meilisearch : typo-tolerance, filtres facettés, suggestions et autocomplétion.
📊 Dashboard auteur
KPIs de téléchargements, gestion des versions, clés API pour l'intégration CI/CD.

Stack technique

CoucheTechnologies
BackendRust, Axum 0.8, Tokio, Tower HTTP 0.6, SQLx
FrontendNext.js 16, React 19, Tailwind CSS 4, TypeScript 5
DonnéesPostgreSQL 16, Meilisearch v1.7
InfraDocker Compose, GitHub Actions, Cloudflare R2

Démarrage rapide

Consultez le guide Démarrage rapide pour installer et lancer l'environnement de développement en quelques minutes. 

git clone https://github.com/FabLrc/pumpkin-hub.git
cd pumpkin-hub
docker compose up --build

Services disponibles :

  • Frontend — http://localhost:3000
  • API — http://localhost:8080/api/v1/health
  • Meilisearch — http://localhost:7700
