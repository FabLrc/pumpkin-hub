Design System

Brutaliste Industrielle — Le langage visuel de Pumpkin Hub.

Philosophie

Le design system de Pumpkin Hub s'inspire de l'esthétique "Brutaliste Industrielle" : une interface sans artifice qui reflète la modernité et la rigueur du langage Rust. Chaque décision visuelle suit ces principes :

  • No-Radius — Aucun arrondi. Les éléments ont des angles droits exclusivement.
  • Code-First — L'interface est pensée pour les développeurs : monospace omniprésent, données techniques en évidence.
  • 60-30-10 — 60% noir (fond), 30% gris neutre (texte), 10% orange (actions/CTA).
  • Clarté technique — Pas de décoration superflue. Chaque pixel sert la compréhension.

Grille & espacement

Composants

Boutons

Deux variantes principales, toujours avec border-radius: 0 :

  • Primary — Fond orange (#f97316), texte noir, hover → #ea580c
  • Ghost — Fond transparent, bordure neutre, texte gris, hover → bordure claire

Cartes (Bento)

  • Bordure 1px #262626, fond #111
  • Hover : bordure → #f97316, translateY(-2px)
  • Pas d'ombre portée, pas d'arrondi

Badges

  • Police JetBrains Mono, 0.6rem, uppercase
  • Bordure colorée, fond semi-transparent (5% opacity)
  • Pas d'arrondi

Inputs

  • Fond #111, bordure #262626
  • Focus : bordure → #f97316, glow subtil
  • Police JetBrains Mono pour les champs techniques

Effets visuels

Noise overlay

Texture de bruit SVG très subtile (opacity 3%) appliquée sur le body::before en position: fixed avec pointer-events: none.

Scanlines

Effet de lignes de scan CRT via repeating-linear-gradient (amplitude 4px, opacity 3%).

Glow orange

Box-shadow subtil sur les éléments en focus ou feature : box-shadow: 0 0 40px rgba(249, 115, 22, 0.15).

Grid Background

Grille orthogonale de fond (48px) en orange très léger (3% opacity) pour renforcer l'esthétique technique/industrielle.
