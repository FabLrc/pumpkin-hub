Pumpkin Hub

Le registre communautaire de plugins et ressources pour le serveur Minecraft Pumpkin MC, propulsé par Rust.

Important
Pumpkin Hub est en phase de construction intensive. Les interfaces et APIs sont susceptibles d'évoluer rapidement. Cette documentation reflète l'état actuel du projet.

Présentation

Pumpkin Hub est la plateforme centrale dédiée à la distribution et la découverte d'extensions pour le serveur Minecraft Pumpkin. Parce que Pumpkin redéfinit la performance serveur avec Rust, Pumpkin Hub redéfinit la gestion de contenu avec une interface minimaliste, une sécurité accrue des binaires et une expérience développeur sans friction.

Contrairement aux plateformes Java classiques (SpigotMC, Modrinth), Pumpkin Hub est conçu nativement pour l'écosystème Rust : gestion des binaires compilés, vérification d'intégrité SHA-256, support multi-architectures CPU (x86, ARM) et intégration GitHub native.

Fonctionnalités clés

Recherche instantanée
Propulsée par Meilisearch pour des résultats en millisecondes, avec filtres par catégorie, architecture et version.
Crate-Centric
Gestion native des binaires compilés Rust et des architectures CPU cibles (x86_64, aarch64).
Sécurité native
Signature SHA-256 systématique et vérification d'intégrité de chaque fichier distribué.
Dependency Graph
Visualisation claire des dépendances entre plugins pour garantir la compatibilité serveur.
Dashboard auteur
Analytics de téléchargements, gestion des versions et publication directe depuis GitHub.
Interface brutaliste
Design "No-Radius" industriel optimisé pour la clarté technique et la performance.

Audiences cibles

Développeurs (Crate Authors)

Publication fluide de plugins, intégrée à GitHub, avec gestion des versions compilées, des dépendances et des architectures CPU cibles. Le dashboard offre des analytics détaillés sur les téléchargements et l'utilisation.

Administrateurs (Server Owners)

Catalogue ultra-rapide avec vérification d'intégrité SHA-256 et informations claires sur la compatibilité des versions pour garantir la stabilité du serveur Pumpkin.

Communauté

Plateforme open-source et transparente, conçue pour favoriser l'adoption et la croissance de l'écosystème Pumpkin MC.
